# AEGISOS Sentinel Event Specification

All perception events generated by Sentinel observers conform to the normalized event schema:

```json
{
  "id": "evt_1784572800000_a1b2c3",
  "timestamp": "2026-07-21T15:48:34.855Z",
  "source": "sentinel",
  "observer": "file_observer",
  "category": "filesystem",
  "priority": "medium",
  "payload": {
    "eventType": "add",
    "filePath": "C:\\Users\\ishar\\Projects\\SecondBrain\\00_Inbox\\quick_capture.md",
    "filename": "quick_capture.md",
    "extension": ".md"
  },
  "version": "1.0.0",
  "correlationId": "corr_1784572800000"
}
```

---

## Event Categories & Event Types

| Category | Observer | Event Types |
| :--- | :--- | :--- |
| `filesystem` | `file_observer` | `add`, `change`, `unlink` |
| `vault` | `vault_observer` | `NOTE_CREATED`, `NOTE_UPDATED`, `NOTE_DELETED` |
| `workspace` | `workspace_observer` | `WORKSPACE_OPENED`, `WORKSPACE_CHANGED` |
| `git` | `git_observer` | `GIT_BRANCH_CHANGED`, `GIT_COMMIT_DETECTED` |
| `clipboard` | `clipboard_observer` | `CLIPBOARD_UPDATED` |
| `system` | `system_observer` | `SYSTEM_TELEMETRY_UPDATED` |
