# AEGISOS v1.0.0 SDK Reference Guide

AEGISOS exposes programmatic SDK interfaces for runtime integration.

---

## Published SDK Facades

- **Tool SDK**: `toolRuntimeAPI`
- **Agent SDK**: `agentRuntimeAPI`
- **Workflow SDK**: `workflowAPI`
- **Memory SDK**: `memoryAPI`
- **Governance SDK**: `governanceAPI`
- **Automation SDK**: `automationAPI`
- **Context & World SDK**: `contextAPI`
- **Knowledge SDK**: `knowledgeAPI`
- **Planner SDK**: `plannerAPI`
- **Simulation SDK**: `simulationAPI`
