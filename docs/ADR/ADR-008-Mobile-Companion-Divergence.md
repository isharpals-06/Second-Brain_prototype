# ADR-008: Form Factor Divergence — Desktop Workstation vs Mobile Companion
**Status:** ACCEPTED  
**Date:** July 23, 2026  

## Context
Shrinking a desktop multi-panel cockpit shell onto mobile creates an unusable, cluttered interface.

## Decision
Diverge UI form factors:
- Desktop = Multi-panel AI Workstation.
- Mobile = AI Companion (Voice interface, Current Objective progress ring, Approval Checkpoints, Quick Capture).

## Consequences
- Delivers a native experience optimized for mobile context.
