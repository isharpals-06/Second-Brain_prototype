# ADR-002: Adaptive Workspace Modes Replacing Screen Routing
**Status:** ACCEPTED  
**Date:** July 23, 2026  

## Context
Page-centric routing models view applications as static document screens.

## Decision
Replace pages with **6 Cognitive Operating Modes** (`OBSERVE`, `THINK`, `RESEARCH`, `BUILD`, `REVIEW`, `FOCUS`). The workspace adapts fluidly depending on human intent and AI state.

## Consequences
- Eliminates page transition lag.
- Focuses UI around human cognitive intent rather than software modules.
