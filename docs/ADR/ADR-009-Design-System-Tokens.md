# ADR-009: Design System Tokens & Sharp Edge Mandate
**Status:** ACCEPTED  
**Date:** July 23, 2026  

## Context
Generic rounded cards and soft drop shadows contradict the aerospace telemetry aesthetic.

## Decision
Enforce strict 0px border radiuses, 1px solid structural borders (`#434655`), obsidian surface levels (`#0A0A0C`), and dual font typography (`Inter` + `JetBrains Mono`).

## Consequences
- Guarantees high-density, engineering-grade visual consistency.
