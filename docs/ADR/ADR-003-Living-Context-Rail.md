# ADR-003: Living Context Rail for Ambient Presence
**Status:** ACCEPTED  
**Date:** July 23, 2026  

## Context
Heavy multi-level navigation sidebars consume screen real estate and obscure context.

## Decision
Implement a slim 64px **Living Context Rail** with vertical active indicator strokes, icon triggers, and ambient AI status lights.

## Consequences
- Navigation becomes an ambient awareness bar.
- Maximizes central workspace real estate.
